# AI Infrastructure Roadmap (বাংলা)

বাংলা ভাষায় AI Infrastructure Engineering শেখার জন্য একটি সম্পূর্ণ Open Source Learning Repository।

এই রিপোজিটরির লক্ষ্য শুধুমাত্র Python শেখানো নয়, বরং ধাপে ধাপে এমন একজন Engineer তৈরি করা, যে Production-ready AI Systems, LLM APIs এবং AI Inference Infrastructure তৈরি ও পরিচালনা করতে পারবে।

এই Roadmap মূলত Intermediate Python Developers-দের জন্য, যারা AI Backend, LLM Infrastructure অথবা AI Platform Engineering-এ ক্যারিয়ার গড়তে চান।

## কেন এই Repository?

আজকাল AI নিয়ে প্রচুর রিসোর্স থাকলেও বেশিরভাগই Prompt Engineering, RAG বা Fine-tuning পর্যন্ত সীমাবদ্ধ।

কিন্তু বাস্তবে একটি AI Application Production-এ চালাতে হলে আরও অনেক কিছু জানতে হয়, যেমন—

- Advanced Python
- FastAPI
- Linux
- Docker
- Kubernetes
- Redis
- PostgreSQL
- GPU Serving
- Distributed Systems
- Model Deployment
- Monitoring
- AI Inference

এই Repository-তে আমরা সেই বিষয়গুলোই শুরু থেকে ধাপে ধাপে শিখবো।

## এই Repository-তে কী পাবেন?

- Structured Learning Roadmap
- বাংলা ভাষায় সহজ ব্যাখ্যা
